# Gerador de Senhas

Aplicativo Web, para ser usado como um gerador de senhas.

Desenvolvido em JAVA e JSF.
- NetBeans IDE 8.2
- Java Server Faces (Prime Faces)
- Hibernate 4.3.1
- Apache Tomcat 8.0.27.0
- Java EE 7 Web
- Repositório Maven

•	Instalado Hibernate para persistência na base de dados.
•	Instalado JavaServerFaces para recurso gráfico Prime Faces.
•	Ativado e atualizados índices Maven para gerenciar dependências.
•	Startado Servidor TomCat para subir aplicação, porta 8084 localhost.

## Configuração necessária para funcionamento:

1. Baixar o projeto, executar o build na máquina na IDE NetBeans, aguardar browser abrir.
2. Possui persistência, mas não é necessário utilizar a base, criado no ínicio do projeto somente para testes.
3. Maven responsável pela imortação de todas as dependências.


## Funcionalidades

•	Tela Inicial, opções: GERENTE/ CLIENTE / ATENDIMENTO.
•	Tela GERENTE, opções: Chamar nova senha ou Reiniciar gerador.
•	Tela CLIENTE, opções: preferencial ou normal. (Verificar regra de negócio)
•	Tela de ATENDIMENTO:visualizar senha prefencial.
